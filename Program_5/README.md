PRE-REQUESTIES:-
    MAKE SURE THAT YOU HAVE INSTALLED Node AND Mongodb

Step 1:
    Create a new folder and name it whatever you want

Step 2:
    Then open the terminal and follow the following instructions
        1. npm i express mongodb

Step 3:
    Create files for the following
        1. index.html 
        2. index.js

Step 4:
    Open your terminal and use the following to run your server
        cd "Your-Folder-Name"
        node index.js

Step 5:
    Head over to the browser you are familar with and use the following link to visit you locally hosted website
        http://localhost:3000/
